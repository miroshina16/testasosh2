# Currency Converter - Конвертер валют

## Автор
** Осинина Анна** 

## Описание
GUI-приложение «Book Tracker» для управления списком прочитанных книг с возможностью фильтрации, сохранением данных в JSON и использованием Git.

## Функционал
- Выбор книги
- Ввод автора
- Выбор жанра
- Таблица истории всех операций
- Сохранение истории в JSON
- Загрузка истории из файла
- Валидация ввода (только положительные числа)

## Как получить API-ключ
1. Перейдите на [exchangerate-api.com](https://app.exchangerate-api.com/sign-up)
2. Зарегистрируйтесь (бесплатный план - 1500 запросов/месяц)
3. После регистрации скопируйте API-ключ из панели управления
4. Вставьте ключ в файл `main.py` (строка `API_KEY = "ваш_ключ"`)
